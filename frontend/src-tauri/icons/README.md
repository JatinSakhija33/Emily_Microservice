# App Icons for ATSN AI Desktop App

This directory should contain the following icon files for the Tauri desktop application:

## Required Icons:

### PNG Files (cross-platform):
- `32x32.png` - 32x32 pixel icon
- `128x128.png` - 128x128 pixel icon
- `128x128@2x.png` - 256x256 pixel icon (high DPI)

### macOS:
- `icon.icns` - macOS icon bundle (can be created from PNG using tools like IconMaker)

### Windows:
- `icon.ico` - Windows icon file (can be created from PNG using online converters)

## How to Create Icons:

1. Design or obtain a square icon for "ATSN AI" (preferably with the logo/text)
2. Use tools like:
   - [IconMaker](https://iconmaker.creator-spring.com/) for macOS .icns
   - [Online ICO converter](https://favicon.io/favicon-converter/) for Windows .ico
   - Resize PNGs to required dimensions

## Notes:
- All icons should have transparent backgrounds where possible
- Keep the design consistent with the web app branding
- Place files directly in this `icons/` directory
